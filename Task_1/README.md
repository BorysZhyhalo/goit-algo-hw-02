# Homework 2 - Task 1 (Queue)

## Опис

Ця програма імітує роботу сервісного центру:

- автоматично створює нові заявки;
- додає їх у чергу `Queue` з модуля `queue`;
- послідовно обробляє заявки (FIFO: first in, first out).

Реалізація відповідає псевдокоду з умови:

- `generate_request()` - створення та додавання заявки в чергу;
- `process_request()` - видалення заявки з черги та "обробка";
- головний цикл - повторює кроки, доки користувач не введе `exit`.

## Вимоги

- Python 3.8+ (рекомендовано 3.10+)

## Як запустити

1. Переконайтесь, що файл програми збережений, наприклад, як `main.py`.
2. У терміналі відкрийте папку проєкту.
3. Запустіть:

```bash
python main.py
```

## Як користуватись програмою

Після запуску програма показує запрошення:

- натисніть `Enter` - виконати наступний цикл;
- введіть `exit` - завершити роботу.

Кожен цикл виконує:

1. `generate_request()` - створює заявку з унікальним ID та додає в чергу;
2. `process_request()` - дістає заявку з черги і позначає як оброблену;
3. `time.sleep(1)` - невелика пауза для імітації часу обробки.

## Приклад роботи

```text
Програма запущена. Введіть 'exit' для завершення.
Натисніть Enter для наступного кроку або 'exit':
Заявка #1 створена та додана в чергу.
Заявка #1 оброблена.
----------------------------------------
Натисніть Enter для наступного кроку або 'exit': exit
Програму завершено.
```

## Структура даних

Використано `Queue` з модуля `queue`, що гарантує обробку заявок у правильному порядку:

- перша додана заявка обробляється першою;
- нові заявки додаються в кінець черги.
