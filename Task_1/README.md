# Homework - Task 1 (Queue)

## Опис

Ця програма імітує роботу сервісного центру з приймання та обробки заявок.

Програма дозволяє:

- створювати нові заявки;
- додавати заявки в чергу `Queue` з модуля `queue`;
- послідовно обробляти заявки за принципом FIFO;
- перевіряти кількість заявок у черзі;
- завершувати роботу програми командою `exit`.


## Реалізація

У програмі реалізовано такі основні частини:

- `generate_request()` — створює нову заявку з унікальним ID та додає її в чергу;
- `process_request()` — дістає першу заявку з черги та позначає її як оброблену;
- `show_queue_size()` — показує кількість заявок, які зараз знаходяться в черзі;
- `main()` — головний цикл програми, який очікує команди користувача.

## Як користуватись програмою

Після запуску програма очікує введення команд користувача.

Користувач може:

- створити нову заявку;
- обробити першу заявку з черги;
- перевірити кількість заявок у черзі;
- завершити програму.

Програма працює за принципом черги **FIFO**:

> First In, First Out — перша додана заявка обробляється першою.

## Доступні команди

| Команда | Опис |
|---|---|
| `create` | Створити нову заявку та додати її в чергу |
| `process` | Обробити першу заявку з черги |
| `status` | Показати кількість заявок у черзі |
| `exit` | Завершити роботу програми |

## Структура даних

У програмі використано клас `Queue` з модуля `queue`.

Черга працює за принципом FIFO:

- нові заявки додаються в кінець черги;
- першою обробляється та заявка, яка була додана найраніше.

Наприклад:

```text
create  -> Заявка #1
create  -> Заявка #2
create  -> Заявка #3

process -> обробиться Заявка #1
process -> обробиться Заявка #2
process -> обробиться Заявка #3
```
