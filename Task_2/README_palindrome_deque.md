# Завдання 2 — Перевірка паліндрому за допомогою deque

## Опис завдання

Необхідно розробити функцію, яка приймає рядок як вхідний параметр, додає всі його символи до двосторонньої черги `deque` з модуля `collections`, а потім порівнює символи з обох кінців черги, щоб визначити, чи є рядок паліндромом.

Програма повинна:

- використовувати `deque` з модуля `collections`;
- перевіряти, чи є рядок паліндромом;
- коректно працювати з рядками парної та непарної довжини;
- бути нечутливою до регістру;
- ігнорувати пробіли;
- ігнорувати розділові знаки.

## Що таке паліндром?

Паліндром — це слово, фраза або рядок, який читається однаково зліва направо і справа наліво.

Наприклад:

```text
madam
level
A man, a plan, a canal, Panama
No 'x' in Nixon
```

## Реалізація

У програмі використовується функція:

```python
def is_palindrome(s):
```

Вона виконує такі кроки:

1. Приймає рядок `s`.
2. Переводить текст у нижній регістр.
3. Видаляє пробіли та розділові знаки.
4. Додає очищені символи у `deque`.
5. Порівнює символи з лівого та правого краю черги.
6. Повертає:
   - `True`, якщо рядок є паліндромом;
   - `False`, якщо рядок не є паліндромом.

```python
collections
```

## Як користуватись програмою

У поточній реалізації програма вже має готовий список тестових рядків:

```python
test_cases = [
    "A man, a plan, a canal, Panama",
    "No 'x' in Nixon",
    "Hello, World!",
    "Was it a car or a cat I saw?",
    "Madam In Eden, I'm Adam",

]
```

Щоб перевірити інший рядок, можна додати його до списку `test_cases`.

```python
test_cases = [

```

## Приклад роботи програми

Після запуску програми у терміналі буде виведено результат перевірки для кожного рядка:

```text
'A man, a plan, a canal, Panama' -> True
'No 'x' in Nixon' -> True
'Hello, World!' -> False
'Was it a car or a cat I saw?' -> True
'Madam In Eden, I'm Adam' -> True
'Not a palindrome' -> False
```

## Приклад ручного тестування

Можна перевірити функцію окремо:

```python
print(is_palindrome("level"))
print(is_palindrome("hello"))
print(is_palindrome("Madam"))
print(is_palindrome("A man, a plan, a canal, Panama"))
```

Очікуваний результат:

```text
True
False
True
True
```

## Очікуваний результат

Програма повинна правильно визначати паліндроми незалежно від:

- великих або малих літер;
- пробілів;
- ком;
- апострофів;
- інших розділових знаків.

Наприклад, рядок:

```text

```

після очищення перетворюється на:

```text

```

Цей рядок читається однаково зліва направо і справа наліво, тому функція повертає:

```text
True
```
